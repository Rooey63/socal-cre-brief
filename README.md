# SoCal CRE Daily Brief

Automated daily commercial real estate market analysis for Southern California.

## Features
- Daily email with market synthesis (6 AM PST)
- Real-time dashboard with market metrics
- Data from major CRE sources and brokers
- REIT valuations and economic indicators

## Dashboard
View the live dashboard: https://rooey63.github.io/socal-cre-brief/

## Data Sources
- BizNow
- LA Business Journal
- San Diego Union-Tribune
- CBRE, Cushman & Wakefield, JLL, Newmark, Colliers
- Federal Reserve (Treasury, SOFR)
- Alpha Vantage (S&P 500, REITs)
